# Security

Synesis is an early developer preview, not a production-secure system. Do not
use it with credentials, private keys, sensitive data, or untrusted artifacts.
The threat model and known limitations are documented in
[`docs/security/THREAT_MODEL.md`](docs/security/THREAT_MODEL.md).

## Reporting

Do not put credentials, private keys, exploit details, private network data, or
session transcripts in public issues. If GitHub private vulnerability
reporting is enabled for this repository, use it. Otherwise, report privately
through the repository owner's authenticated GitHub account and do not create
a public issue containing sensitive details.

There is no security-response SLA for this preview. Please include the commit,
affected component, reproduction steps that contain no secrets, and impact
assessment when reporting a problem.
